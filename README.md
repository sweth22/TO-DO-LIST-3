📌 Smart Task Board – DOM Manipulation Portfolio Project
📖 Overview

Smart Task Board is a simple and interactive task management web application built using HTML, CSS, and JavaScript (DOM Manipulation). It allows users to add tasks, manage progress visually, and switch between themes for a better user experience.

This project demonstrates core DOM manipulation concepts, event handling, dynamic UI updates, and responsive design.

✨ Features
➕ Add new tasks dynamically
📋 Display tasks in a clean task board
📊 Progress bar to track completion progress
🌙 Toggle between Light and Dark themes
🎨 Responsive and modern interface
⚡ Real-time DOM updates without page reload
🛠️ Technologies Used
HTML5 – Structure
CSS3 – Styling and responsive layout
JavaScript (DOM Manipulation) – Functionality and interactions
📂 Project Structure
Smart-Task-Board/
│
├── index.html      # Main HTML file
├── dom.css         # Stylesheet
├── dom.js          # JavaScript functionality
└── README.md       # Project documentation
🚀 How to Run the Project
Method 1: Open Directly
Download or clone the project.
Open index.html.
Double-click the file to launch in your browser.
Method 2: Using VS Code
Open project folder in VS Code.
Install Live Server extension.
Right-click index.html.
Select Open with Live Server.

These projects are interactive web applications built using HTML, CSS, and JavaScript, focused on practicing DOM Manipulation concepts.

Both projects are beginner-friendly and designed to strengthen core front-end development skills.

📋 Display tasks in a dynamic list
